== Usage ==

Add the Mean, Median, Mode Calculator shortcode to your page, post or sidebar:

`[calcio_mean_median_mode_calculator]`

Add the following code to your template where you would like the Mean, Median, Mode Calculator to appear:

`<?php display_calcio_mean_median_mode_calculator(); ?>`
